**VidyaBox** is a free, student-focused resource hub designed to make academic materials easily accessible.  
Browse and download **university papers, syllabi, and textbooks** — all in one place, with a clean interface and motivational quotes.

---

## ✨ Features
- 📖 Easy access to academic PDFs (papers, syllabi, textbooks, notes)
- 🔍 Search bar to quickly find resources
- 🎯 Motivational quotes that refresh on each visit
- 📱 Responsive design for mobile and desktop
- 🎨 Simple, student-friendly UI with warm colors
